Python clase de estadística:

para la libreria numpy
np.median(): Calcula la mediana de un array.
np.mean(): Calcula la media aritmética de un array.
np.var(): Calcula la varianza de un array.
np.std(): Calcula la desviación estándar de un array.
Parámetro ddof es un parámetro opcional, que significa "degrees of freedom" (grados de libertad).

para la libreria statistics
variance(): Calcula la varianza de un array.
stdev(): Calcula la desviación estándar de un array

- statistics es ideal para cálculos estadísticos simples y pequeños conjuntos de datos.
- numpy es preferible cuando trabajamos con grandes conjuntos de datos, debido a su rendimiento y eficiencia en cálculos vectorizados.
